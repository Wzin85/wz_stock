# WZ Stock 대화 및 작업 요약

마지막 업데이트: 2026-06-13

## 프로젝트 목적

미국 주식의 단기 스윙 매매를 돕는 프로그램이다.

- 스크리너가 S&P 500 종목에서 후보를 기계적으로 발굴한다.
- 기존 앱이 기술 지표, 시장 상황, 보유 상태를 정리한다.
- Claude 같은 LLM은 모든 종목을 무작정 스캔하는 대신, 선별된 후보의
  위험·근거·예외 상황을 종합 판단하는 역할을 맡는다.
- 최종 매매 결정은 사용자가 한다.
- 백테스트는 LLM 판단을 포함하지 않고 추천 후보를 규칙대로 모두
  매매했을 때의 기본 전략 성능과 새로운 팩터를 검증한다.

## 우리가 합의한 방향

1. 보기 좋은 UI보다 판단 근거와 데이터 품질을 우선한다.
2. 스크리너는 빠르고 저렴한 1차 필터이며 Claude를 종목마다 호출하지 않는다.
3. LLM은 정량 규칙을 대체하기보다 정량 신호 이후의 맥락 판단에 사용한다.
4. 전략 변경은 한 번에 하나씩 하고, 같은 기간과 같은 seed로 비교한다.
5. 한 기간에서만 좋아진 결과는 채택하지 않고 독립 기간에서도 검증한다.
6. 현금이 오래 놀지 않는 방향을 선호하지만, 무리한 진입으로 MDD가
   커지는 것도 함께 경계한다.
7. 백테스트의 목적은 성과 확인뿐 아니라 새로운 유효 팩터 발굴이다.

## 앱과 스크리너

- React/Vite 앱이며 GitHub Pages로 배포한다.
- Twelve Data로 OHLCV를 받고 Claude Sonnet으로 종합 분석한다.
- GitHub Gist로 보유 종목, 설정, 분석 기록, 스크리너 결과를 기기 간 동기화한다.
- 차트, 지지·저항, 포지션 사이징, 일봉·주봉 분석이 들어 있다.
- Claude 판단 자료에 상대강도, RSI 다이버전스, 거래량 패턴,
  52주 위치 등이 포함된다.

스크리너:

- S&P 500 고정 목록에서 워치리스트·보유 종목을 항상 포함하고 나머지는
  랜덤 표본으로 채운다.
- Mode A 추세추종과 Mode B 역추세반등 후보를 분리한다.
- GitHub Actions가 평일 한국시간 15:30에 실행할 수 있다.
- 결과를 Gist로 올리고 Slack 알림을 보낸다.
- 휴대폰을 계속 켜둘 필요가 없다.

## 백테스트 공통 규칙

- yfinance 데이터와 로컬 Parquet 캐시를 사용한다.
- 현재 S&P 500 구성 종목을 과거로 소급하므로 생존편향이 남아 있다.
- 매 시점에는 그날까지의 데이터만 사용한다.
- 신호는 종가 확정 후 만들고 다음 거래일 시가에 매수한다.
- 초기자금 $5,000, 최대 5종목, 종목당 균등 배분이다.
- 매수·매도 각각 비용 0.1%를 반영한다.
- 손절은 진입가에서 2 ATR, 목표는 4 ATR이다.
- 최대 보유기간은 20거래일이다.
- Mode A 룰 깨짐은 종가가 MA50 아래로 내려갈 때다.
- Mode B 룰 깨짐은 RSI가 60을 넘을 때다.
- SPY 종가가 MA200 아래이면 Mode A 신규 매수를 중단하고 Mode B는 유지한다.
- 여러 랜덤 150종목 표본을 반복해 평균과 편차를 확인한다.

## 현재 채택된 기본 전략 v6

### Mode A

- MA20 > MA50
- 현재가 > MA20
- MA20이 5거래일 전보다 상승
- RSI 50~70
- 거래량비율 1.3 이상 또는 10일 거래량가중 CLV 0.2 초과
- 같은 날 후보가 많으면 MA20 이격률 4%에 가까운 순서로 선택

이 MA20 이격률 선택 방식은 2016~2018 독립 기간에서도 기존 방식보다
수익과 Mode A 성과가 좋아져 v5 기본값으로 채택했다.

### Mode B

기존 5개 조건 중 4개 이상:

- RSI 35 이하
- 볼린저밴드 하단 근접
- 52주 고점 대비 -25%~-8%
- 거래량비율 1.5 이상
- 현재가가 MA50의 97% 이상

Mode B 조건 자체는 아직 v1을 유지하고 있다.

### v6 청산 규칙

v6부터 종가 기준 ATR 트레일링 스탑을 기본으로 채택했다.

- 기존 초기 손절: 진입가 - 2 ATR
- 기존 목표가: 진입가 + 4 ATR
- 종가가 진입가 + 1 ATR에 도달하면 활성화
- 이후 손절가: 보유 중 최고 종가 - 2 ATR
- ATR은 진입 신호일 값을 고정 사용
- 손절가는 내려가지 않고 올라가기만 함
- 오늘 종가로 계산한 stop은 다음 거래일부터 적용
- 기존 룰 깨짐과 20일 timeout은 유지

기존 v5 청산 방식은 `--no-trailing-stop` 옵션으로 재현할 수 있다.

### Mode B ATR 5% 상한

Mode B 신규 진입에서 ATR(14)/종가가 5% 이상인 후보를 제외했다.

- 아직 기본 v5에는 적용하지 않았다.
- `--mode-b-atr-max 5` 실험 옵션으로 유지한다.

### Mode B 상대강도

앱과 같은 방식으로 계산한다.

- 1개월 RS = 종목 21거래일 수익률 - SPY 21거래일 수익률
- 3개월 RS = 종목 63거래일 수익률 - SPY 63거래일 수익률

기존 v5 거래 분석에서 Mode B의 유력 구간은 1개월 RS -10%~-5%였다.

- 고유 신호 1,256건
- 승률 51.1%
- 거래당 평균 +0.72%
- 중앙값 +0.35%

의미:

- Mode B에서는 시장보다 강한 종목보다 SPY보다 적당히 뒤처진 눌림 종목이
  반등 후보로 더 적합할 가능성이 있다.
- 너무 강한 RS +5% 이상과 지나치게 약한 구간은 불안정했다.
- 2019~2021 고유 신호 성과는 약해 기간 의존 가능성이 있었다.

검증한 실험:

- 같은 날 Mode B 후보를 1개월 RS -7.5% 근접순으로 재정렬
- 옵션: `--selection ma20-gap-rs1m`
- 기존 v5와 같은 네 기간·같은 seed로 비교

결과:

- 네 기간 평균수익: +16.27% → +16.30%로 사실상 동일
- 평균 MDD: -22.12% → -22.40%로 소폭 악화
- Mode B 승률: 48.7% → 약 48.4%
- Mode B 평균수익: +0.45% → 약 +0.35%
- 2019~2021은 개선됐지만 2016~2018과 2023~2025는 악화

결론:

- RS -7.5% 근접순은 기계적 Mode B 선택 규칙으로 채택하지 않는다.
- 상대강도는 Claude 종합 판단의 참고자료로는 계속 사용한다.
- `ma20-gap-rs1m`은 연구 재현용 옵션으로만 남긴다.
- 기본값은 계속 `ma20-gap`이다.

### Mode B MA50 충족 우선순위

Mode B 조건 자체는 바꾸지 않고, 같은 날 후보 중 MA50근접 조건을 충족한

결론:

- `ma20-gap-ma50`은 연구 재현용 옵션으로만 유지한다.

## VIX 팩터 분석

VIX를 전략에 적용하기 전에 기본 v5의 다섯 기간 거래에 신호일 VIX를
결합해 상관관계를 분석했다.

추가된 연구 지표:

- VIX 절대값
- 1일·5일·20일 변화율
- 20일·60일·252일 백분위
- 20일 고점 대비 하락률
- 전일 하락과 10일 고점 후 첫 하락 신호

핵심 결과:

- Mode A의 VIX 252일 백분위 80~100:
  1,293개 고유 신호, 승률 58.5%, 평균 +1.67%, 중앙값 +1.71%
- Mode A의 VIX 252일 백분위 60 이상은 다섯 기간 평균이 모두 양수였다.
- Mode B의 VIX 252일 백분위 60~80:
  813건, 승률 54.0%, 평균 +1.35%, 중앙값 +1.17%
- Mode B의 80~100 극단 공포는 2019~2021 평균 -2.93%로 불안정했다.
- 단순 10일 VIX 고점 후 첫 하락은 Mode B 전체보다 나빴다.
- 5% 이상 첫 하락은 전체 성과가 좋았지만 2022~2024에서 실패했다.
- VIX 5일 -20% 미만 급락은 Mode B에서 좋았지만 표본이 129건으로 작다.

해석:

- 절대 VIX보다 장기 백분위가 더 유용했다.
- Mode A는 강세장 안의 높은 공포 조정 구간이 유력하다.
- Mode B는 극단 공포보다 중상위 공포 구간이 더 안정적일 가능성이 있다.
- 아직 어떤 VIX 조건도 기본 전략에 적용하지 않았다.

## Mode A 고VIX 우선 배정 실험

- Mode A 거래 수: 16,937 → 18,337
- Mode A 승률: 49.12% → 49.91%
- Mode A 평균수익: +0.588% → +0.673%

- Mode B 승률: 50.89% → 48.29%
- Mode B 평균수익: +0.722% → +0.324%

- 기본 전략에 채택하지 않는다.

## 종가 기준 ATR 트레일링 스탑 실험

기존 손절과 목표는 유지하고, 종가가 진입가보다 1 ATR 이상 상승하면
보유 중 최고 종가 - 2 ATR로 손절가를 올렸다. 신호일 ATR을 고정 사용하며,
오늘 종가로 계산한 새 손절가는 다음 거래일부터 적용한다.

- 실험 버전: `CLOSE_TRAILING_ACTIVATE_1ATR_DISTANCE_2ATR_v1`
- 옵션: `--trailing-stop`
- 다섯 기간 평균수익: +20.23% → +25.91%
- 평균 MDD: -20.08% → -18.65%
- 평균 손익비: 1.33 → 1.43
- 평균 승률: 49.72% → 49.20%
- 실행 수익 표준편차 평균: 16.64% → 18.61%
- 동일 seed 100개 중 개선 54회, 악화 46회
- timeout 비율: 38.13% → 25.07%
- 총 거래 수: 23,213 → 25,516

기간별 수익은 다섯 기간 모두 개선됐고 MDD는 네 기간에서 개선됐다.
Mode A/B 모두 평균 거래수익이 소폭 좋아졌다. 승률 개선이 아니라 이익 보호와
빠른 자금 회전이 성과 개선의 주된 경로다.

결론:

- 계획한 검증에서는 반복성, 손익비, MDD, timeout 기준을 대체로 충족했다.
- 그러나 seed 승패가 54 대 46으로 약하고 실행 편차는 증가했다.
- 즉시 기본 v5를 교체하지 않고 `v6 후보`로 분류한다.
- 기본값은 off이며 연구 옵션으로 재현 가능하다.

보고서:
`backtest/results/comparison_v5_close_trailing_stop.txt`

### 트레일링 독립 검증 2009~2012

기존 실험과 겹치지 않는 2009~2012 기간과 신규 seed 20개로 검증했다.

- 평균수익: +23.40% → +35.61%
- 평균 MDD: -20.73% → -18.42%
- 평균 손익비: 1.30 → 1.46
- 승률: 49.5% → 48.5%
- 실행 수익 표준편차: 23.83% → 17.91%
- 동일 seed: 개선 14회, 악화 6회
- seed별 수익 변화 중앙값: +13.8%p
- timeout 비율: 40.74% → 27.12%
- Mode A 평균수익: +0.42% → +0.51%
- Mode B 평균수익: +1.05% → +1.28%

결론:

- 독립 기간에서도 수익, MDD, 손익비, timeout, seed 반복성이 개선됐다.

- 현재 S&P 500 종목을 과거로 소급하므로 오래된 기간일수록 생존편향이 크다.
- 다음 파라미터 민감도 검증을 통과하면 v6 기본 채택을 검토한다.

### 트레일링 파라미터 민감도

독립 기간 2009~2012에서 활성화 기준과 트레일링 간격을 한 축씩 변경했다.

- 활성화 0.75 ATR: 수익 +37.59%, MDD -18.58%, seed 16승 4패
- 활성화 1.00 ATR: 수익 +35.62%, MDD -18.42%, seed 14승 6패
- 활성화 1.25 ATR: 수익 +34.67%, MDD -19.22%, seed 14승 6패
- 간격 1.50 ATR: 수익 +35.19%, MDD -18.76%, seed 16승 4패
- 간격 2.50 ATR: 수익 +28.33%, MDD -19.73%, seed 10승 10패
- 트레일링 없음: 수익 +23.51%, MDD -20.73%

모든 주변값이 기본 v5보다 수익과 MDD에서 개선됐다. 1.0/2.0은 최고 수익
조합은 아니지만 수익, MDD, 손익비, timeout의 균형이 좋고 탐색 구간의
중앙값이므로 보수적인 기본값으로 선택했다.

결론:

- 단일 숫자 과최적화 가능성이 낮아졌다.
- 트레일링 `1.0/2.0`을 백테스트 기본 전략 v6에 채택했다.
- `--trailing-activate-atr`, `--trailing-distance-atr`로 연구 재현 가능하다.
- `--no-trailing-stop`으로 기존 v5를 재현할 수 있다.

보고서:
`backtest/results/sensitivity_trailing_2009_2012.txt`
`backtest/results/sensitivity_trailing_review.txt`

## 바로 이어서 할 작업

1. Mode B VIX 60~80 적용 방식을 v6 트레일링 기준에서 다시 설계한다.
2. VIX는 일별 공통값이므로 종목 정렬이 아니라 Mode B 진입 허용량이나
   자금 배정 규칙으로 검증한다.
3. VIX 80 이상 Mode B는 제거보다 후순위 대체 후보로 남긴다.
4. ATR 5% 상한은 별도의 위험관리 후보로 계속 보류한다.

## 해석 원칙

- 팩터 분석의 상관관계가 실제 포트폴리오 개선을 보장하지 않는다.
- 전체 수익이 좋아져도 해당 Mode 자체가 나빠졌다면 원인을 다시 확인한다.
- 평균뿐 아니라 중앙값, 손절률, MDD, 기간별 반복성, seed 편차를 함께 본다.
- 백테스트 결과는 과거 성과이며 미래 수익을 보장하지 않는다.
